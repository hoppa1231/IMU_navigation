Rollout CSV: `derived/predictions/dataflash_rollout/imu_att_h5000_l5000_rollout.csv`
HTML: `artifacts/generated/dataflash_rollout/index.html`

Method: within each rolling-validation fold, use non-overlapping prediction rows only. Start from the true current POS at the first selected row, then accumulate predicted 5-second displacement vectors.

This is a sparse displacement rollout, not a full inertial navigation integration at IMU rate.
