# Reports Layout

This directory is split by role.

## Active structure

- `reports/navigation/`:
  current GPS/POS, flow, IMU, POLI_NA, and comparison reports.
- `reports/final/`:
  current final summaries for the project and the best DataFlash candidate.
- `reports/experiments/`:
  generated experiment reports for model sweeps, rolling validation, and rollout variants.

## Historical root files

The remaining Markdown files in the root of `reports/` are older baselines,
notes, and work logs kept for traceability. New navigation/final outputs should
not be written there anymore.
