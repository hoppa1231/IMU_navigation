# Generated artifacts

This directory contains files generated by project scripts. They are not original flight logs or source datasets.

Original GPS-capable inputs currently inspected:

- `derived/dataflash/GPS.csv`
- `derived/dataflash/POS.csv`
- `artifacts/linear_15_01_2025.csv`
- `artifacts/triangle_15_01_2025.csv`
- `artifacts/data.csv`

Generated GPS visualizations are under `artifacts/generated/gps_flights/`.

`artifacts/data.csv` is split into `module_data/segment_01` through `module_data/segment_07` because its `TimeStamp` resets indicate seven concatenated flights.
